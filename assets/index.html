HTML

  • head contains information, or metadata, that helps the browser render the page correctly.
  • Metadata is data about a webpage that the browser uses to effectively serve page to users.
  • meta element defines metadata such as character set, description, keywords, author, and viewport.
  • viewport sets the default view of a page.
  • Semantic HTML Elements: the name of the element describes its content and its role on the page.
  • Element is the entire element from opening tag to closing while tag is only what is inside the brackets.

CSS

  • There are inline, internal, or external CSS. External CSS is most common.
  • Separation of Concerns: each section of code should have its own responsibility.
  • display: block assigns each image to start on a new line.
  • box shado: x and y offsets relative to the element, blur and spread radius, and color.

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
